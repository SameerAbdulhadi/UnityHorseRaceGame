Works with URP, HDRP and Standard render pipelines. By default it comes with URP render pipeline materials.

For use in HDRP: Edit ->Rendering -> Materials -> Convert all built-in materials to HDRP. After that select Both Directional lights in the sceneview to get the correct light settings.
For use in URP: Edit ->Rendering -> Materials -> Convert all built-in materials to HDRP. After that select Both Directional lights in the sceneview to get the correct light settings.
